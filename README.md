Tools collection for Tsinghua University.

Make a symlink ``thu``(or whatever you like) on your PATH, point it to the ``run`` script.

Available commands:

    thu net
    thu net login
    thu net logout

    thu usereg iplist
    thu usereg ipup x.x.x.x
    thu usereg ipdown x.x.x.x
    thu usereg ipdown 0/1/2

    thu user setuser
